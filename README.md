# analisis-etiquetas-redes
Exploración y normalización de etiquetas Tema/Subtema en datos de redes sociales
Este repositorio reúne material de un trabajo académico orientado a explorar y normalizar etiquetas de tema y subtema en datos de menciones en redes sociales.
Objetivos del trabajo:
 identificar redundancias y superposiciones en etiquetas
 proponer criterios de normalización
 apoyar la organización y análisis de los datos
Herramientas utilizadas:
 Python
Pandas
